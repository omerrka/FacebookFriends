# FacebookFriends

#### A simple application that lists and provides personal details of your friends.

## Features

#### * MVVM design pattern
#### * Programmatic UI
#### * Offline usage
#### * Protocol Delegate Pattern



## How to Use

#### Before starting the application, follow the steps below.

`pod repo update`

`pod install`
